**Discord Sample MMO**

某MMOくんリスペクトの自作BOTに似た挙動するサンプルコードです<br>
5時間半で組みました<br>
prefixは勝手に変えといてください<br>

!atk / !attack ... 敵に攻撃<br>
!wait ... 何もしない<br>
!item ... アイテム<br>
!item <アイテム名> <対象(i,t,e使用時)(任意,無ければ自身へ)> ... アイテムを使用<br>
!reset / !re ... リセット<br>
!fix ... 詰んだ時に停止判定を解除するやつ<br>
!status / !st ... ステータス
